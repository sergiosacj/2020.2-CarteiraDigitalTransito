    Backlog do Produto

    1. Introdução

       O backlog do produto é uma técnica geralmente utilizada no desenvolvimento ágil. É basicamente uma lista com breves descrições de funcionalidades ou desejos necessários para alcançar um objetivo. Essa lista traz os requisitos para um projeto, priorizados de acordo com o valor entregue para o cliente. Desse modo, por meio desse documento será possível ter uma visão mais ampla do produto.

    2. Metodologia

       Criaremos uma tabela no seguinte molde:

    Épico História de Usuário Rastreabilidade Ator Desejo Objetivo Prioridade
    Cadastro US01 RF8 Motorista Realizar cadastro no CDT Ter acesso à CNH virtualmente Alta

       Desse modo, teremos catalogado todos os requisitos, com todas as suas prioridades, mais todos os épicos e histórias de usuário.
